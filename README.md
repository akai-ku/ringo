# 🍎 ringo — 併読ノート

同時に読み進めている本(併読)を管理するための、シンプルなWebアプリです。
バーコード(ISBN)をカメラで読み取って本を登録できます。

`eigo.html` には姉妹アプリ「[英文暗記](#-英文暗記-eigohtml)」があります(写真から英文を取り込んで暗記・発音練習)。

## 機能

- **バーコード読み込みで本を登録**
  - 本の裏表紙のバーコード(978 / 979 で始まるISBN)をカメラでスキャン
  - ブラウザの `BarcodeDetector` API を利用し、非対応ブラウザでは [ZXing](https://github.com/zxing-js/library) に自動フォールバック
  - カメラが使えない環境向けに ISBN の手入力にも対応(ISBN-10 / ISBN-13 どちらも可)
- **書誌情報の自動取得**
  - [openBD](https://openbd.jp/)(日本の書籍に強い)→ [Google Books](https://developers.google.com/books) の順で検索し、タイトル・著者・出版社・表紙・ページ数を自動入力
- **併読の管理**
  - 読書中 / 中断中 / 読了 のステータス管理
  - ページ数による進捗バー
  - 読書中リストは「最後に読んでから時間が経っている本」ほど上に表示され、7日以上触れていない本にはバッジが付くので、読み残しに気づけます
  - メモの記録、手動追加・編集・削除
- **データはブラウザの localStorage に保存**(サーバー不要)
- **Googleスプレッドシートへのバックアップ・同期**(任意)
  - ページを開くと自動で最新データを読み込み、「💾 保存」でスプレッドシートに保存
  - 端末の買い替えや複数端末での利用に対応。セットアップ手順は [docs/sheets-sync-setup.md](docs/sheets-sync-setup.md) を参照

## 使い方

静的ファイルのみで動作します。ビルドは不要です。

```bash
# 例: Python の簡易サーバーで起動
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

> **注意**: カメラ(バーコードスキャン)は、セキュリティ上 `https://` または
> `http://localhost` でのみ動作します。スマートフォンで使う場合は
> GitHub Pages などの HTTPS 環境に配置してください。

## 🗣️ 英文暗記 (eigo.html)

写真で撮った英文を暗記するためのアプリです。スラッシュリーディング教材
(`A: Good morning, / Mr. Tanaka. // Can I ask you / …` のような
`/` `//` でチャンク区切りされた会話文)にそのまま対応しています。

- **📷 写真から英文を取り込み**
  - 英語ページを撮影すると [Tesseract.js](https://github.com/naptha/tesseract.js) がブラウザ内でOCRし、`A:`/`B:` の発話ごとに1枚のカードにします(1行 = 1カード。読み取り結果は登録前に編集可)
  - 日本語訳のページも撮影すると、行ごとに英文と対応づけて訳として登録できます
  - 訳がない行は [MyMemory API](https://mymemory.translated.net/) の自動翻訳で補えます(あとで編集可)
  - 写真が横向きのときは「↻ 回転」で直してから読み取れます
- **🧠 間隔反復で暗記**
  - 「もう一度 / できた / 余裕」の自己採点で次の復習日が自動調整されます(翌日 → 3日後 → 1週間後 → …)
  - 「今日の復習」だけが出題されるので、毎日ひらくだけでOK
- **🇯🇵→🇬🇧 日→英変換練習**: 日本語訳を見て英語で言えるかを試すモード(逆の英→日モードもあり)
- **🎤 発音チェック**
  - 🔊 お手本の読み上げ(ゆっくり再生も可)— Web Speech API
  - 🎤 マイクに向かって話すと音声認識で聞き取り、単語ごとに一致・不一致を色分けして採点
- データは併読ノートと同じくブラウザの localStorage に保存されます

> 読み上げ・発音チェックはブラウザの Web Speech API を使うため、
> Chrome / Safari での利用をおすすめします。カメラ・マイクは
> `https://` または `http://localhost` でのみ動作します。
